# DevPulse Frontend

This is the frontend application for DevPulse, an interactive workspace for engineering teams. 
It connects to the DevPulse backend and provides a seamless UI for querying federated data across GitHub, Linear, Slack, and Sentry.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **Components:** React (with Recharts for data visualization)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
